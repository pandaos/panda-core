# Panda Core

System backend and start session and more.

## Dependencies

```shell
sudo pacman -S pkgconf cmake qt5-base qt5-x11extras qt5-tools kidletime polkit polkit-qt5 pulseaudio
```

## Build

```shell
mkdir build
cd build
cmake ..
make
```

## Install

```shell
sudo make install
```
